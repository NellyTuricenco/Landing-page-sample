Описание:
В файле index.html содержится готовая HTML-структура. По всему файлу есть комментарии, которые должны дать понять, какой именно элемент нужно выбрать в таблице style.css, чтобы к нему применилась стилизация.
В файле index.html абсолютно ничего нельзя менять (или добавлять)!

В файле style.css содержится частично готовая CSS-стилизация. Необходимо заменить все ??? на нужный селектор, пользуясь описанием из комментария (расположенного над ???), а также комментариями из файла index.html (в некоторых ситуациях). Помимо этого, в файле style.css есть комментарии с заданиями, для которых не написано решение. Его нужно написать самостоятельно. В некоторых комментариях будет сразу несколько заданий. Для их решения может понадобиться создать несколько селекторов. Дополнять уже существующую стилизацию можно, но для этого нужно отделять новые стили дополнительным переносом строки в области правил, чтобы сразу было видно, что именно добавилось. Удалять существующую стилизацию нельзя.

При выполнении МОЖНО:
- пользоваться селекторами тегов и атрибутов, а также любыми псевдо-селекторами
- дополнять существующую стилизацию, отделяя добавленные правила одним переносом строки

При выполнении НЕЛЬЗЯ:
- удалять комментарии к заданиям
- удалять или добавлять что-либо в файл index.html
- менять структуру файлов и папок

При выполнении НУЖНО:
- заменить в таблице стилей ??? на нужный селектор, пользуясь описанием (комментарий) над ним
- добавить необходимую стилизацию там, где сказано это сделать
